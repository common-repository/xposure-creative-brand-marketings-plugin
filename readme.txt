== Description ==

This free plugin from Xposure, makes accessing creative new ideas for your online and offline marketing, easier than ever.  The Xposure plugin includes UK social media statistics, the best SEO tips for your business and how to use Twitter in your digital marketing, amongst many more great articles.  Using CSS3, everything looks great, functions beautifully and makes these marketing morsels easy to digest!  <a href="http://www.creativebrandmarketing.co.uk/invigorate-marketing-plans-handy-litt">Read more about this great new plugin here</a>.

== Installation ==

1. Upload `xposure-creative-brand-marketings-plugin` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
